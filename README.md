# Hosting
Firebaseを用いて[Webページ](https://myfirstlp.web.app)を公開しています。

# Todo機能
- ログイン認証（Firebase Authentication）
- 保存・削除（Cloud Firestore, LocalStrage）

# 使用技術
- HTML/CSS
- JavaScript （Todoリスト, スクロール, ヘッダー監視など）
- Sass
- jQuery (ナビスクロール・ドロップダウン)
- Chart.js
- React.js, Slack通知 (お問い合わせ)
- Git, Github
- Firebase 9.23.0
